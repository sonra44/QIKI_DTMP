# Отчёт: proto_extensions/utils/proto_linter.py

## Назначение
Простой линтер для проверки `.proto` файлов.

## Задачи
- Проверяет использование поля `scalar_data` внутри блока `oneof`.
- Убеждается, что поля `status_code` объявлены как перечисления (`enum`).
- Формирует список предупреждений и выводит количество найденных проблем.

## Комментарии
Анализирует файлы из каталога `protos`. Правила можно расширять добавлением новых функций проверки.
