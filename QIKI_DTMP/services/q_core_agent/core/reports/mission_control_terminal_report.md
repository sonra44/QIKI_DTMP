# mission_control_terminal.py — Анализ задач

## Основное назначение
Терминал Mission Control обеспечивает удалённое управление космическим аппаратом через интерфейс `prompt_toolkit`.

## Ключевые задачи
- Инициализация основных модулей корабля и контроллеров.
- Ведение журнала событий и вычисление времени миссии.
- Обновление телеметрии и проверка уровня тревоги в фоне.
- Формирование графического интерфейса: заголовок, панели телеметрии и логов.
- Обработка команд оператора и горячих клавиш (автопилот, аварийная остановка, диагностика).
- Асинхронный запуск приложения и корректное завершение работы.
