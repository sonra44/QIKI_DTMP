# proposal_evaluator.py — Анализ задач

## Основное назначение
ProposalEvaluator отбирает наиболее подходящие предложения на основе приоритета и уверенности.

## Ключевые задачи
- Загрузка порога уверенности из конфигурации.
- Сортировка входных предложений по типу и приоритету.
- Отбрасывание вариантов ниже порога уверенности.
- Выбор лучшего предложения и формирование итогового списка.
- Логирование принятых и отклонённых предложений.
