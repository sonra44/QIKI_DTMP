# qiki_mission_control_pro.py — Анализ задач

## Основное назначение
Профессиональная версия Mission Control с двуязычным интерфейсом и реалистичной симуляцией параметров.

## Ключевые задачи
- Инициализация корабельных систем и управление состоянием миссии.
- Поддержка двуязычности и нескольких представлений (SUMMARY, SYSTEMS, NAV и др.).
- Фоновая симуляция телеметрии и генерация случайных событий.
- Обновление миссионных данных и журнала событий.
- Формирование интерфейса `prompt_toolkit` с горячими клавишами и обработкой команд.
- Поддержка автопилота через контроллер логики.
