# Отчет: qiki_mission_control.py

## Назначение
Базовый терминал Mission Control, работающий в чистом ASCII режиме.

## Основные задачи
- Инициализация `ShipCore` и контроллеров актюаторов и логики.
- Живое обновление навигационных и сенсорных данных.
- Фоновая симуляция параметров с возможностью автопилота.
- Ведение журнала событий и команд.

## Возможные улучшения
- Добавить тесты для функций автопилота и обработки команд.
- Рассмотреть поддержку асинхронных операций.
