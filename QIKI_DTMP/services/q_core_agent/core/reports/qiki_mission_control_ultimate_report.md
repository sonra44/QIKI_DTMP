# Отчет: qiki_mission_control_ultimate.py

## Назначение
Интерактивный терминал управления миссией с использованием `prompt_toolkit`.

## Основные задачи
- Инициализация корабельных систем через `ShipCore`, `ShipActuatorController` и `ShipLogicController`.
- Поддержка живой телеметрии, журналирования событий и параметров миссии.
- Фоновая симуляция и автопилот с периодическим обновлением состояния.
- Логирование и вывод статуса подключения к космическому аппарату.

## Возможные улучшения
- Расширить обработку ошибок при установке зависимостей.
- Добавить дополнительные сценарии автопилота и событий.
