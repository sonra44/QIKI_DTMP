# Отчет: rule_engine.py

## Назначение
Модуль правил для генерации предложений на основе состояния систем.

## Основные задачи
- Проверка статуса BIOS и предложение перехода в SAFE_MODE при ошибках.
- Формирование структуры `Proposal` с командой для системного контроллера.
- Логирование активированных правил.

## Возможные улучшения
- Добавить больше правил, учитывающих телеметрию и состояние FSM.
- Реализовать конфигурируемые приоритеты правил.
