# Отчет: ship_actuators.py

## Назначение
Высокоуровневый контроллер исполнительных систем космического корабля.

## Основные задачи
- Определение осей тяги и режимов движения.
- Управление основным двигателем и RCS через команды `ActuatorCommand`.
- Распределение энергетических ресурсов между подсистемами.
- Логирование выполненных операций и текущего режима.

## Возможные улучшения
- Добавить проверки диапазонов значений и обработку исключений для всех актюаторов.
- Реализовать обратную связь о состоянии команд.
