# Отчет: ship_bios_handler.py

## Назначение
Обработка BIOS и диагностика всех подсистем корабля.

## Основные задачи
- Запуск комплексной диагностики корпуса, питания, двигателей, сенсоров, жизнеобеспечения и вычислительных модулей.
- Формирование отчета `BiosStatusReport` с детализацией проблем и показателем здоровья.
- Логирование состояния систем и выявленных неисправностей.

## Возможные улучшения
- Уточнить коды ошибок и расширить диагностику датчиков.
- Добавить поддержку пользовательских профилей диагностики.
