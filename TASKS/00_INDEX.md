# TASKS (execution cabinet)

Цель: держать **каждую задачу отдельным MD** рядом с кодом, но без создания второй доски “Now/Next”.

## Канон приоритета

- Единственная доска приоритетов: `~/MEMORI/ACTIVE_TASKS_QIKI_DTMP.md`
- Этот каталог `TASKS/` — **детализация и исполнение**, а не список “что сейчас важнее”.

## Как создавать задачу

1) Создай новый файл по шаблону: `TASKS/TEMPLATE_TASK.md`
2) Имя файла: `TASK_YYYYMMDD_SHORT_SLUG.md`
3) Добавь ссылку на задачу из `~/MEMORI/ACTIVE_TASKS_QIKI_DTMP.md` (в виде пути `TASKS/...`), но **не копируй** блоки Now/Next/Backlog сюда.

## Правила (анти-дубли)

- В задаче разрешены локальные чеклисты “что проверить”, команды, вывод, ссылки на файлы/коммиты.
- Запрещено: заводить второй “канонический план” приоритетов (Now/Next) в `TASKS/`.
- Исторические отчёты/исследования — оставляем как есть; если они вводят в заблуждение как “current”, добавляем пометку “historical”.
