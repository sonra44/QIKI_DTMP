# Отчёт по файлу `services/q_core_agent/config/ship_config.json`

## Содержимое
Большая конфигурация корабля: корпус, энергетические системы, двигатели, сенсоры, жизнеобеспечение и вычислительные модули.

## Задачи
- [ ] Проверять соответствие `ship_id` и `ship_class` фактическому кораблю.
- [ ] Актуализировать параметры корпуса и энергетики при модернизации.
- [ ] Контролировать распределение мощности в `power_distribution`.
- [ ] Добавить мониторинг топлива и состояния реактора.
- [ ] Расширять список сенсоров и их статус при обновлениях.
- [ ] Поддерживать актуальность данных систем жизнеобеспечения и вычислительных модулей.
