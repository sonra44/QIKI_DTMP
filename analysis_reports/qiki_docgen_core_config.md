# Отчёт по файлу `tools/qiki_docgen/core/config.py`

## Назначение
Модуль управления конфигурацией `qiki-docgen`. Загружает YAML-файл, определяет корень проекта и предоставляет доступ к путям и параметрам генерации.

## Основные задачи
- Определение корневой директории проекта и поиск файла конфигурации.
- Парсинг YAML с помощью встроенного метода `_parse_yaml` без внешних зависимостей.
- Предоставление значений по умолчанию при отсутствии или ошибке загрузки конфигурации.
- Утилиты для получения путей, include-списка `protoc` и произвольных параметров (`get_path`, `get_protoc_includes`, `get`).

## Возможные улучшения
- Заменить самописный парсер на использование `yaml.safe_load` для надёжности.
- Добавить валидацию структуры конфигурации и сообщения об ошибках.
- Реализовать возможность динамического обновления настроек без перезапуска.
