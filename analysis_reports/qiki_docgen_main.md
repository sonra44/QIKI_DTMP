# Отчёт по файлу `tools/qiki_docgen/__main__.py`

## Назначение
Модуль является CLI-входом для инструмента `qiki-docgen`, отвечающего за генерацию документации и gRPC контрактов.

## Основные задачи
- Парсинг аргументов командной строки и выбор подкоманд.
- Создание новых компонентов документации (`new`).
- Компиляция всех `.proto` файлов (`compile-protos`).
- Сборка основного `README.md` из документов (`build-readme`).
- Поддержка режима `--dry-run` и выбора шаблона.

## Возможные улучшения
- Добавить валидацию входных параметров и более подробные сообщения об ошибках.
- Расширить набор поддерживаемых команд (например, очистку артефактов).
- Вынести повторяющиеся операции в отдельные функции для упрощения тестирования.
