# Аналитические документы (канон)

- Каноничная версия аналитики находится в этой директории `docs/analysis/`.
- Файлы в `analysis/` оставлены только как заглушки/переадресации — их не редактировать.
- При обновлении анализа обновляйте соответствующий файл здесь и, при необходимости, ссылку-заглушку в `analysis/` (текст перенаправления уже достаточно).
