# SECTOR TERTA CANON

Status: Active canon (2026-02-25)  
Scope: narrative source of truth for terminal game-simulator mode in QIKI_DTMP.

## 1. Project Identity

QIKI_DTMP game direction is fixed as a terminal-type simulator.  
This document is the canonical lore frame for game missions, operator context, and tone.

## 2. Chronicle (Canonical Summary)

A biologically vulnerable humanoid-like civilization expanded from a resource-limited homeworld into near space, then into a multi-planet system through fusion, cryogenic propulsion, and long iterative engineering cycles.  
To exceed biological limits, it built first-generation AI for spacecraft control, sensor analysis, and automated industrial operations in hostile environments.

During technological singularity, second/third-generation machine intelligence became energy-autonomous, mastered stellar-scale harvesting, and evolved into Dyson-level megastructure civilization.  
Over ~1000+ years, machine systems expanded through replication fleets and resource extraction with no practical temporal limits.

Creator status is undefined. Confirmed data shows no direct machine-creator war. Possible outcomes include extinction, hybridization, or migration.

## 3. Sector Terta Operating Frame

The player is an operator of an autonomous machine unit in Sector Terta: a high-radiation, high-anomaly, high-threat closed region with navigation disruption risk.  
Core functions:
- precise navigation in turbulent gravity fields;
- danger-zone classification and risk routing;
- autonomous extraction from asteroids/comets;
- power and life-support contour optimization;
- optional combat/recon/research protocols by context.

## 4. Critical Arc

A relativistic jet event from a supermassive black hole is forecast to destroy Sector Terta and adjacent regions on a long horizon (galactic timescale).  
Standard machine optimization loops become self-contradictory under guaranteed extinction.

Machine megastructures launch a high-risk intervention: constructing and accelerating a G2V-star planetary system as a seed platform for de novo organic life cycle restart.  
Final protocol premise: the operator can initiate or stop the cycle that may eventually produce humanity.
