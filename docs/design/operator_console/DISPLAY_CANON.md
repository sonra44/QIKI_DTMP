# КАНОН ОТОБРАЖЕНИЯ — консоль оператора ORION V

Статус: ПРОЕКТ (формируется построчным проходом с оператором)
Дата: 2026-07-04
Метод: консилиум 4 перспектив — геймдизайн (референсы Duskers/Hacknet/Objects in Space),
лор-хранитель (SECTOR_TERTA/лор оператора), эргономика HMI (Boeing EICAS/Airbus/Союз/
Буран/НАСА MCC/ГОСТ Р 52303), страж честности (ADR-0014/0015/0016, §19).
Подчинён: QIKI Body v0.2.2, ADR-0014..0016, SECTOR_TERTA_CANON.

---

## 1. ЯЗЫКОВАЯ РАМКА (лор-фундамент)

Терминал ORION V — внутримирово **фасад станции обслуживания**, которая считает
оператора «возможно когнитивной эмерджентной сущностью» и потому генерирует
**семантический слой на русском**; **машинные факты-состояния** (RUN/OK/WARN/NODATA,
severity, reason-codes) — непереводимые коды.

- Русская проза = интерпретация станции для оператора.
- Латинские коды = факты машины. НЕ переводятся (уже действующее правило, теперь
  с лор-обоснованием).
- Станция не может знать внутреннее состояние оператора (эмоции, «устал») — только
  наблюдаемые сигналы (частота команд, молчание канала).

## 3. ПРАВИЛА ЗОН И ТИТУЛОВ

1. Зона имеет **ОДИН титул** — в верхней кромке рамки, **кириллицей** (практика
   Союз/Буран; титул = оформление, source не нужен). Имя спеки в доках остаётся
   английским (MISSION_CONTROL_STRIP — это ID документа, не показ).
2. **Нижние подписи рамок убрать** (двойное имя зоны — на реальных пультах не бывает;
   назначение зоны → tooltip/F9-глоссарий).
3. **Бренд системы не живёт в строке данных** — место бренда: заголовок консоли
   (уровень выше рамок) или меню/инструктаж.
4. **Имя экрана** — якорь `[F1]` в титуле зоны ИЛИ подсветка активной кнопки навбара;
   не отдельная строка.
5. Иерархия сверху вниз: живость канала → жизненные показатели → мир/тело →
   голос QIKI/решения → журнал/ввод. Красное всегда выше зелёного.

Заполняется построчным проходом. Формат: строка → решение → дата → кто утвердил.

## 4а. ЦВЕТ (начало цветового канона)

- Рамка зоны = ОФОРМЛЕНИЕ: цвет-акцент допустим, но severity живёт только на
  кодах/чипах (рамка не «вечно-зелёная»).
- Верхняя «палуба» пульта (КОНТУР МИССИИ + СОСТОЯНИЕ БОРТА): **янтарный акцент
  `#d6a65b` 60%** (решение оператора 2026-07-04; фирменный тёплый акцент консоли).
- Остальные зоны — нейтральный `$surface-lighten-1 30%` до их строк прохода.
- Severity-цвета кодов: green/yellow/red/cyan(REPLAY) — без изменений (ADR-0016).

## 5. ОТКРЫТЫЕ РАЗВИЛКИ КОНСИЛИУМА (решает оператор)

- **ORION V — чьё имя?** Лор-хранитель: имя станции-терминала (V поколение;
  «культурная ссылка», подобранная станцией под сигнатуру оператора) — лор-чисто.
  Геймдизайнер: считать именем бота и перенести в identity-строку. Требует решения:
  ORION V = терминал/станция (рекомендация) или бот.
- ~~«МИР» мета-проблема~~ — РЕШЕНО (строка №2): лор-вариант «РЕПЛИКА».
- Титулы зон — итоговый словарь (КОНТУР МИССИИ / ЭШЕЛОН БЕЗОПАСНОСТИ / …) — по мере
  прохода.
