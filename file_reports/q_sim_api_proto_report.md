# Отчет по файлу `protos/q_sim_api.proto`

## Краткое описание
Файл определяет gRPC-сервис `QSimAPI`, используемый для взаимодействия между агентом Q-Core и сервисом симуляции. Содержит RPC-вызовы для получения данных сенсоров, отправки команд актуаторам и проверки состояния сервиса.
