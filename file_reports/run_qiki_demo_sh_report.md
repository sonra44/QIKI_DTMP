# Отчет по файлу `scripts/run_qiki_demo.sh`

## Краткое описание
Скрипт запускает сервисы `q_sim_service` и `q_core_agent` в фоне, записывая их вывод в отдельные лог-файлы. Предназначен для демонстрации работы системы.
