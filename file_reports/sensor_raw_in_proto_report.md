# Отчет по файлу `protos/sensor_raw_in.proto`

## Краткое описание
Файл описывает сообщение `SensorReading` для передачи показаний различных сенсоров. Структура поддерживает идентификатор сенсора, тип, временную метку и данные в одном из нескольких форматов.
