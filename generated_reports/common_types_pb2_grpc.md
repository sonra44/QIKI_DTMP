# Отчёт по файлу `generated/common_types_pb2_grpc.py`

## Задачи модуля
- Выполняет проверку версии установленного пакета `grpcio` относительно версии, использованной для генерации.
- Не содержит описаний сервисов, поскольку в `common_types.proto` отсутствуют gRPC-определения.

## Примечание
Файл автоматически создан плагином gRPC для Python.
