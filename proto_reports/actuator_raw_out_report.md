# Отчёт по файлу `actuator_raw_out.proto`

## Задачи
- Определяет сообщение `ActuatorCommand` для управления исполнительными механизмами.
- Поддерживает разные типы значения команды через `oneof` (`float`, `int`, `bool`, `Vector3`).
- Хранит идентификаторы команды и исполнительного механизма, а также временную метку.
- Включает перечисление `CommandType` с типами команд (скорость, поворот, включение и т.д.).
- Предоставляет параметры исполнения: единицы измерения, уверенность, таймаут, требование подтверждения и число повторов.
