# Отчёт по файлу `bios_status.proto`

## Задачи
- Описывает состояние устройств через сообщение `DeviceStatus` с идентификатором, именем и статусами.
- Определяет перечисления `Status`, `DeviceType` и `StatusCode` для классификации состояния и типа устройства.
- Сообщение `BiosStatusReport` агрегирует результаты POST, сведения о прошивке и здоровье системы.
- Содержит метрики: показатель здоровья, время последней проверки и аптайм.
