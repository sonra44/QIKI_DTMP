# Отчёт по файлу `common_types.proto`

## Задачи
- Предоставляет базовые типы, используемые другими протоколами.
- Определяет сообщение `UUID` для уникальных идентификаторов и структуру `Vector3` для трёхмерных координат.
- Содержит перечисления `SensorType`, `ActuatorType` и `Unit` для описания типов оборудования и единиц измерения.
