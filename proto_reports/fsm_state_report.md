# Отчёт по файлу `fsm_state.proto`

## Задачи
- Стандартизует состояния и переходы конечного автомата через перечисления `FSMStateEnum` и `FSMTransitionStatus`.
- Сообщение `StateTransition` фиксирует переходы между состояниями с указанием события и статуса.
- `FsmStateSnapshot` хранит текущее состояние, историю переходов и дополнительный контекст.
