# Отчёт по файлу `proposal.proto`

## Задачи
- Описывает сообщение `Proposal` для представления предложений действий системы.
- Хранит идентификатор предложения, источник, временную метку и список действий `ActuatorCommand`.
- Предоставляет дополнительные сведения: обоснование, приоритет, ожидаемую длительность и метаданные.
- Содержит перечисления `ProposalType` и `ProposalStatus`, а также зависимости, конфликты и подпись предложения.
