# Контракт события: `qiki.events.v1.system_mode` (v1)

## Назначение

`qiki.events.v1.system_mode` — событие, которое публикуется QIKI при смене режима системы.

Режим — это **системное состояние**, а не “роль промпта”. ORION только отображает режим.

## Версионирование

- `event_schema_version` — версия payload-схемы.
- Для текущей версии: `event_schema_version == 1`.

## Формат payload (v1)

См. JSON Schema: `payload.schema.json`.
