# Отчет по файлу `ship_core.py`

## Задачи
- Управление идентификацией корабля: генерация и хранение уникального идентификатора.
- Загрузка конфигурации корабля из JSON-файла и выбор режима работы.
- Предоставление структур данных для статусов систем: корпус, энергоустановка, двигатели, сенсоры, жизнеобеспечение, вычислительная подсистема.
- Методы получения текущего состояния каждой системы на основе конфигурации и динамических данных.
- Регистрация и обработка данных сенсоров, хранение последних показаний.
- Отправка команд актуаторам с проверкой наличия соответствующих систем.
- Формирование сводного отчета о состоянии корабля.
