# Отчет по файлу `ship_fsm_handler.py`

## Задачи
- Определение набора состояний космического корабля и перечисления `ShipState`.
- Формирование контекста `ShipContext` для проверки работоспособности систем, навигации и стыковки.
- Реализация обработчика конечного автомата `ShipFSMHandler`, отвечающего за переходы между состояниями.
- Логика переходов с учетом состояния систем, режима двигателей и наличия цели для стыковки.
- Выполнение аварийной остановки и логирование критических событий.
- Предоставление краткой сводки состояния корабля для диагностики.
