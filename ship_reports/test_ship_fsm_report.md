# Отчет по файлу `test_ship_fsm.py`

## Задачи
- Создание упрощенного логического контроллера для моделирования поведения корабля без protobuf-зависимостей.
- Определение возможных состояний корабля и логики их переходов при разных условиях.
- Симуляция нескольких циклов работы с управлением тягой и маневрами для проверки переходов.
- Вывод результатов каждого цикла и итогового состояния, что позволяет оценить корректность логики.
